Mes notes d'utilisation de `Jekyll` pour créer mon site.

<!--more-->

 - `_posts/` contient les posts, un index des posts est [possible]({% link posts-summary.md %}).
 - `assets/` contient les images et ressources affichées ou téléchargeable.
 - A category can be part of the URL for a post whereas a tag cannot.
 - Drafts are posts without a date in the filename.
 - How to [links](https://jekyllrb.com/docs/liquid/tags/#links) to posts or pages

 - `_config.yml`
   -  Use `excerpt_separator`,
   -  Use `collections` (see [collections](https://jekyllrb.com/docs/collections/))

 - YAML Header
   - `published` set to false to hide post,

 - `bundle exec jekyll serve` pour lancer le serveur en local.
